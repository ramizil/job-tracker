🎯 Job Tracker

A private, local AI job-search assistant — try it out!
עוזר חכם ופרטי לחיפוש עבודה שרץ על המחשב שלך — בואו לנסות!

What is it?  Track every job you apply to and let AI do the heavy lifting: analyze your fit, write cover letters & recruiter notes, tailor your resume, research the company, prep you for interviews, and more. Everything runs on your own computer — your data never leaves your machine.

מה זה?  כלי לניהול חיפוש העבודה שלך, עם AI שעושה את העבודה הקשה: מנתח התאמה למשרה, כותב מכתבים מקדימים והודעות למגייסים, מתאים את קורות החיים, חוקר את החברה, מכין אותך לראיונות ועוד. הכול רץ על המחשב שלך — המידע לא יוצא החוצה.

100% local & private · מקומי ופרטי Windows & macOS Free AI tier (Gemini) · שכבת AI חינמית

⚙️ How to install  ·  איך מתקינים

Before you start  ·  לפני שמתחילים

Install Python 3.10 or newer. On Windows, tick “Add Python to PATH” during install. התקינו Python 3.10 ומעלה. ב-Windows סמנו "Add Python to PATH" בזמן ההתקנה.
(Recommended) Have Chrome or Edge installed for a clean app window & perfect PDFs — most PCs already do. (מומלץ) שיהיה מותקן Chrome או Edge לחלון אפליקציה נקי ול-PDF מושלם — לרוב כבר מותקן.
Get the app: clone the repo with git clone https://github.com/ramizil/job-tracker.git (needs Git + access) — or ask Rami for a zip. השיגו את האפליקציה: בצעו git clone https://github.com/ramizil/job-tracker.git (צריך Git והרשאה) — או בקשו מרמי קובץ zip.

🪟 Windows

  1. Open the job-tracker folder you cloned (or unzipped), e.g. in Documents.
    פתחו את התיקייה job-tracker שעשיתם לה clone (או שפתחתם מ-zip), למשל ב-Documents.
  2. Double-click start.bat.
    לחצו פעמיים על start.bat.
  3. First run installs everything automatically, then the app window opens.
    בהרצה הראשונה הכול מותקן אוטומטית, ואז נפתח חלון האפליקציה.
  4. To stop: click ⏻ Quit in the app, or press Ctrl+C in the console.
    לעצירה: ⏻ Quit באפליקציה, או Ctrl+C בחלון הקונסול.

🍎 macOS

  1. Open the job-tracker folder you cloned (or unzipped).
    פתחו את התיקייה job-tracker שעשיתם לה clone (או שפתחתם מ-zip).
  2. Double-click start.command.
    לחצו פעמיים על start.command.
  3. If macOS blocks it: right-click → Open (once), or run chmod +x start.command in Terminal.
    אם macOS חוסם: קליק-ימני → Open (פעם אחת), או chmod +x start.command בטרמינל.
  4. To stop: click ⏻ Quit, or Ctrl+C in Terminal.
    לעצירה: ⏻ Quit, או Ctrl+C בטרמינל.

One last step — add an AI key  ·  צעד אחרון — הוספת מפתח AI

In the app, open ⚙ Settings, choose a provider and paste an API key. The easiest free option is Google Gemini — get a key at aistudio.google.com. Click Save (no restart needed). באפליקציה, פתחו ⚙ Settings, בחרו ספק והדביקו מפתח API. האפשרות החינמית הקלה ביותר היא Google Gemini — מפתח מקבלים ב-aistudio.google.com. לחצו Save (אין צורך באתחול).
Optional: for job search, add a free JSearch (RapidAPI) key in Settings. רשות: לחיפוש משרות, הוסיפו מפתח חינמי של JSearch (RapidAPI) בהגדרות.
That’s it! The app opens at http://127.0.0.1:5001. Start by clicking 📋 Paste Job and pasting any job ad.
זהו! האפליקציה נפתחת בכתובת http://127.0.0.1:5001. התחילו בלחיצה על 📋 Paste Job והדביקו מודעת דרושים כלשהי.
Tip: Set your own resume in ⚙ Settings → Resume HTML path so the match score and AI features use your CV.
טיפ: הגדירו את קורות החיים שלכם ב-⚙ Settings → Resume HTML path כדי שציון ההתאמה ויכולות ה-AI ישתמשו ב-CV שלכם.
